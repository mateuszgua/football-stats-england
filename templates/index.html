The application shows how to deal with data from csv files.

Below was shown steps of the project: - loading data from source to database with pandas, - using sql queries for reading data from Mysql database, - showing results by charts in matplotlib.


Chapters


  1. Source
    Downloading files from the website and uploading them to the application.
  2. Python - pandas
    Merging all files from sources into one csv file. Making normalization process with pandas. Creating dimensions and fact tables.
  3. Database
    Creating database and tables in MySQL with Python scripts. Creating connections with database and checking data in MySQL Workbench.
  4. Flask
    Creating a web application. Running queries from flask app to MySQL database and loading data to dataframes.
  5. Visualization
    Creating sql queries and riding data from MySQL databse. Creating procedure in sql which allows users choose season and team. Creating visualizations with python and matplotlib.

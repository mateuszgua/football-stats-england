The application shows how to deal with data from csv files.

The main topic is how to load data from source to database with pandas. Also application consider used queries for get data from Mysql database. The last point is shown by the visualization using matplotlib.


Chapters


  1. Source
    Downloading files from the website and uploading them to the application.
  2. Python - pandas
    Merging all files from sources into one csv file. Making normalization process with pandas. Creating tables and save it as csv files.
  3. Database
    Creating database and tables in MySQL with Python commands. Creating connections with database and check data into MySQL Workbench.
  4. Flask
    Creating a web application. Sending queries from flask app to MySQL database and get data into dataframes.
  5. Visualization
    Creating SQL queries and get data from MySQL databse. Creating procedure in SQL for load chosen value from user with buttons. Creating visualization with python and matplotlib.
